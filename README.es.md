## Resumen

`ng-hub-ui-avatar` es un componente de avatar para Angular con soporte para imágenes, iniciales, colores automáticos y fallbacks útiles para interfaces de usuario, listados y perfiles.

## Instalación

```bash
npm install ng-hub-ui-avatar
```

## Uso rápido

```typescript
import { AvatarModule } from 'ng-hub-ui-avatar';
```

## Características

- Fallback automático con iniciales.
- Soporte de imágenes remotas.
- Personalización visual mediante inputs y estilos.
- Integración sencilla en aplicaciones Angular modernas.
